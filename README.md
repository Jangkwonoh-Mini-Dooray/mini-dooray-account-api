# mini-dooray-account-api
- 회원 정보 제공 및 회원 상태를 관리하는 Account API 입니다.
  
<br>

# 🛠 기술 스택
- `Back-End` : Java, Spring Boot, Spring Data JPA, Querydsl
- `Database` : MySQL
- `Tool` : IntelliJ IDEA, Github, Notion, Mockito, JUnit, SonarQube
